Данный репозиторий ***служит портфолио*** и демонстрирует навыки работы с системой контроля версий Git.

## О проекте

Проект создан в рамках обучения на платформе [Stepik](https://stepik.org) у автора [«Pragmatic Programmer»](https://stepik.org/users/612208117/teach). 

В основе лежат два курса:

1. [Git + GitHub](https://stepik.org/course/252829)
2. [Git + GitHub. Полный курс](https://stepik.org/course/214865)

> Второй курс расширяет программу первого: добавлен модуль по Markdown и отдельный раздел, посвященный ветвлению и командной работе.

### Результаты обучения 

Сертификаты, пройденные темы и полученные навыки опубликованы на [GitHub Pages](https://arshinov-dev.github.io/git-learning/).

### История развития 

Рекомендую изучить историю коммитов вместе с журналом обучения [`education-log.md`](https://github.com/arushinofu/git-learning/tree/main/playground/education-log.md). 

Это позволит отследить мои эксперименты и прогресс в работе с инструментом.

### Важное о коде 

Обратите внимание на [README](https://github.com/arushinofu/git-learning/tree/main/playground) в папке `playground`. 

Поскольку фокус обучения был направлен на Git, код внутри `playground` является учебным и служит заглушкой.
