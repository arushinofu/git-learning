Портфолио проект, созданный по итогам прохождения курса **«Основы Git и GitHub»** на платформе Stepik.

---

## 📚 О курсе

- **Платформа:** [Stepik](https://stepik.org/course/252829)
- **Название:** Основы Git и GitHub
- **Статус:** ✅ Пройден
- **Сертификат:** [RU](https://stepik.org/cert/3161888) | [EN](https://stepik.org/cert/3161888?lang=en)

---

## 🎯 Цель проекта

1. Закрепить навыки работы с Git на практике
2. Создать портфолио пройденного курса
3. Вывести результат курса в качестве html страницы с описанием пройденного материала

---

## 📁 Структура проекта
```
git-learning/
├── .gitignore                    # Игнорируемые файлы Git
├── index.html                    # Главная страница
├── detail.html                   # Страница с деталями курса
├── README.md                     # О проекте
├── LICENSE.md                    # Лицензия
├── assets/                       # Необходимое для отображения
│   ├── css/                      # Папка со стилями
│       ├──style.css              # Cтили
│   ├── js/                       # Папка со js
│       ├──main.js                # Скрипты
│   ├── images/                   # Папка с картинками
│       ├──avatar.png             # Аватарка
│       ├──certificate_ru.png     # Сертификат на русском
│       ├──certificate_eng.png    # Сертификат на английском
```
